# 🏥 MediTrack — Live Hospital Resource Tracker

Real-time blood bank availability, donor tracking,
and AI-powered emergency guidance.

## 🚨 Problem

During emergencies, families waste 30-60 minutes
calling hospitals manually. No centralized live
system exists for common people in India.

## 💡 Solution

- 🩸 Live blood bank stock by type
- 🧑‍🤝‍🧑 Donor registration & availability
- 📞 One-click call to hospital/donor
- 📍 GPS map with nearest resources
- 🤖 Gemini AI emergency guidance

## 🎯 SDG Alignment

- SDG 3 — Good Health & Wellbeing
- SDG 11 — Sustainable Cities
- SDG 17 — Partnerships for the Goals

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python, Flask
- Database: Firebase Realtime Database
- AI: Google Gemini API
- Maps: Leaflet.js + OpenStreetMap
- Hosting: Firebase Hosting

## 🚀 How to Run

### Backend

cd backend
pip install -r requirements.txt
python seed_data.py
python app.py

### Frontend

Open frontend/index.html in browser
